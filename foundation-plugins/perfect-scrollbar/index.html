Foundation Perfect Scrollbar
Perfect scrollbar plugin for Foundation.

This plugin integrates Perfect scrollbar into Foundation.

Installation

Download

bower install --save foundation-perfect-scrollbar

Install styles

Basic setup

If you are not making use of Foundation's powerful scss, include the required css to your html file.

<link rel="stylesheet" type="text/css" href="bower_components/foundation-sites/dist/css/foundation.min.css">
<link rel="stylesheet" type="text/css" href="bower_components/foundation-perfect-scrollbar/dist/css/foundation-perfect-scrollbar.min.css">
