Foundation Select
Slick, intuitive and styleable select dropdowns.

This plugin makes it easy create styleable select dropdowns in Foundation. 

<label for="select-1">Basic elect</label>
<select id="select-1" data-select>
    <option value="">Choose an option</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    ...
</select>

<label for="select-2">Multiple select</label>
<select id="select-2" data-select multiple size="6">
    <option value="">Choose an option</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    ...
</select>

Installation

With bower: 
bower install --save foundation-select
