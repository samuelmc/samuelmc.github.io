Foundation Select
Slick, intuitive and styleable.

This plugin makes it easy create styleable select dropdowns in Foundation. 

<select data-select>
    <option value="">Choose an option</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    ...
</select>
<select data-select multiple size="6">
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    ...
</select>
