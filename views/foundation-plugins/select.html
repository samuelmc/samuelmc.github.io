Foundation Select
Slick, intuitive and styleable select plugin.

This plugin makes it easy create styleable select dropdowns in Foundation.

<label for="select-1">Basic select</label>
<select id="select-1" data-select>
    <option value="">Choose an option</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    ...
</select>

<label for="select-2">Multiple select</label>
<select id="select-2" data-select multiple size="6">
    <option value="">Choose an option</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    ...
</select>

Installation

Download

bower install --save foundation-select

Setup styles

Basic setup

If you are not making use of Foundation's powerful scss, include the required css to your html file.

<link rel="stylesheet" type="text/css" href="bower_components/foundation-sites/dist/css/foundation.min.css">
<link rel="stylesheet" type="text/css" href="bower_components/foundation-perfect-scrollbar/dist/css/foundation-perfect-scrollbar.min.css">
<link rel="stylesheet" type="text/css" href="bower_components/foundation-select/dist/css/foundation-select.min.css">
<link rel="stylesheet" type="text/css" href="bower_components/font-awesome/css/font-awesome.min.css">

Usage

Just add a data-select attribute to your select tag.

<select data-select>
    <option value="">Choose an option</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    ...
</select>

The plugin will automaticaly initialize the select dropdowns with foundation.

$(document).foundation();

Options

Option Default value Description
data-icon-class 'fa-caret-down' The FontAwesome class for the select triangle icon.
data-placeholder '' Custom placeholder. Uses the value of the placeholder attribute if available when empty.
data-value '' The default value of the select. Does not override the value attribute.
data-mousewheel true If set to true scrolling over the element will change the value if the element has focus in some browsers.
